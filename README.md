# 🎓 Sistema de Cadastro de Aluno em C

Projeto desenvolvido em linguagem C com o objetivo de praticar entrada e saída de dados, manipulação de variáveis e interação com o usuário via terminal.

---

## 📖 Sobre o Projeto

Este programa realiza um cadastro simples de aluno, solicitando ao usuário:

- Nome
- Idade
- Altura
- Matrícula

Após a entrada dos dados, o sistema exibe as informações cadastradas no terminal.

Projeto desenvolvido como parte dos meus estudos em Análise e Desenvolvimento de Sistemas, com foco em lógica de programação e fundamentos da linguagem C.

---

## 🛠️ Conceitos Aplicados

- Declaração de variáveis (`int`, `float`, `char`)
- Vetor de caracteres (string)
- Entrada de dados com `scanf`
- Saída de dados com `printf`
- Manipulação de memória básica
- Estrutura principal `main()`
